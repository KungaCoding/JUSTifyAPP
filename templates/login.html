JUSTify Player

Ever want to listen to music on Spotify, but not want to create a queue?

JUSTify is an app that plays just the songs from the artists you select by

forming a shuffled queue that includes each artists' top 10 songs in random order.


To use:

1.Login or register for a Spotify account.

2. Specify the device connected to Spotify you wish to interact with (webplayer, mobile phone, desktop app, etc.).

3. Select between 1 - 4 different artists (at the moment you must use exact spelling of each artist).

4. Press create playlist.

5. Start listening!
